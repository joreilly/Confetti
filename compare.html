Each row pairs the rendered PNG (raster source of truth) with the editable figma-svg re-rasterized by your browser. The match column is a windowed SSIM (structural similarity) computed live in the page — pre-blurred and downscaled so a half-pixel offset between the two rasterizers barely moves the score, unlike a per-pixel diff. The SVG is aligned to the PNG first (its export padding + root translate are cropped back out, matching the daemon's fidelity harness), so the score reflects real vector drift, not a constant inset. Hybrid stickers' raster crop layers are inlined so their score reflects the full sticker. The override controls above re-apply a knob to the vector and re-score live: raising font scale grows the SVG's <text> while its captured layer boxes stay put — a fixed-geometry sticker can't reflow, so the glyphs overflow, showing why it can't stand in for a re-rendered fontScale; turning off embedded fonts drops the @font-face so the browser substitutes its own face. Rows sort largest-diff-first once scored, so the worst offenders come to the top.

⚠ The match scores read pixels back from a canvas, which the browser blocks when the images can't be read cross-origin — over file:// (opaque origin), or from a host that doesn't send CORS headers for the PNGs. Open this page over http from a CORS-enabled host: the README's htmlpreview link (images come from raw.githubusercontent.com, which allows it) or a local server (python3 -m http.server in the branch).
